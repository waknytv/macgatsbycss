-Images : Utilisez toujours gatsby-plugin-image pour le chargement différé et le redimensionnement automatique.
-Code splitting : Gatsby le fait automatiquement pour chaque page.
-Minification : Activez la minification du CSS/JS via gatsby-plugin-terser.
-Cache : Mettre en cache les assets statiques.
