# BE WEB — Site vitrine

Ce projet est un site vitrine Gatsby pour les artisans du bâtiment en Belgique.

## Prérequis

- Node.js 18+
- npm

## Commandes

```bash
npm install
npm run develop
```

Autres commandes utiles :

```bash
npm run build
npm run serve
npm run clean
```

## Analytics

Pour activer Google Analytics, définissez la variable d'environnement suivante :

```bash
export GATSBY_GA_TRACKING_ID="VOTRE_ID_GA"
```
